# Azurefy
Automation for Azure Foundations CIS Benchmarks 3.0.0


Be sure to install and (optionally) run azure cli login before running script

```
az login
```


Run All Checks

```
Python3 Azurefy.py
```

Run Specific Script

```
Python3 Azurefy.py --script "check script name"
```
